# Turtle Racing Game

A simple interactive racing game built with Python's `turtle` module. Users can place bets on which turtle will win the race.

## Features
- Multiple turtles with different colors
- User can place a bet before the race starts
- Simple graphics and animation using turtle
- Easy to run and understand

## How to Run
1. Make sure you have Python installed.
2. Run the script using:
```bash
python turtle_race.py
